pub enum ViewResponse<K: Deserialize, V: Deserialize> {
    Reduced(ReducedView<V>),
    Unreduced(UnreducedView<K, V>),
}

Contains the response from the CouchDB server as a result of successfully executing a view.

A ViewResponse takes one of two forms, reduced or unreduced, depending on whether the view's reduce function ran. See the CouchDB documentation for more details.

Although ViewResponse implements the Ord and PartialOrd traits, Chill makes no guarantee how that ordering is defined and may change the definition in an upcoming release. Chill defines the ordering only so that applications may use ViewResponse in ordered collections such as trees.
